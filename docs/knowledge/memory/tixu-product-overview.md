Tixu.ai is a mobile-first product (narrow phone-width column even on desktop) with two halves:

1. **Learn** — gamified, Duolingo-style bite-sized interactive lessons on using AI tools (ChatGPT, Claude, Gemini; image/video models Kling, Nano Banana, Flux, Runway). Personalized: AI profile, "Personal focus" (e.g. Claude Mastery), goal (e.g. Make money online), 20 min/day, recommended order. A course is a winding path of lesson nodes → chapters → final test → certificate. Lessons are interactive: bold emoji headings, 3D illustrations, inline quizzes (multiple-choice, yes/no, fill-blank, Hint). Career tracks (AI for Accountants/PMs/HR/Sales) and a "Personal AI Income Challenge".
2. **Use** — the "AI tools" tab is a hub to actually run models (ChatGPT, GPT-5, Gemini, Runway, Flux, Whisper, Wan), Generate text/image/video/audio, a prompt-tips library, and a chat box.

Bottom nav: Home / Library / AI tools. Routes: /home, /library, /ai-tools, /course/:id, /course/:id/topic/:t/lesson/:l. Login is email-first two-step (/auth/signin → /auth/signin/password).

Marketing positioning: "pocket AI coach that turns you from watcher into someone who earns with AI" — learn + use in one app. See [[tixu-brand-tokens]] and [[tixu-motion-project]].
